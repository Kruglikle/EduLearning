# Edu Learning

Промо-сайт образовательной студии с курсами по английскому языку, Blender и AI / NLP / Python.

Проект выполнен как frontend MVP на моковых данных. Он демонстрирует визуальный стиль, каталог программ, страницы направлений, преподавателей, отзывов и форму записи на обучение.

## Возможности

- главная страница образовательной студии;
- каталог из 12 моковых курсов с фильтрацией;
- отдельные страницы курсов;
- страницы направлений и преподавателей;
- отзывы студентов;
- контактная форма записи на консультацию;
- тёмная и светлая темы с сохранением выбора;
- выбор цветовой гаммы: зелёная, оранжевая или голубая;
- адаптивная навигация и мобильная вёрстка;
- плавные hover-эффекты, glassmorphism и неоновые акценты;

## Технологии

- HTML5;
- CSS3;
- Vanilla JavaScript;
- hash routing;
- Lucide Icons через CDN;
- Google Fonts через CDN;
- `localStorage` для сохранения темы.
- FormSubmit для отправки заявок на email.

Проект не использует сборщик, frontend-фреймворк или backend.

## Форма заявки

Форма на странице контактов отправляет заявки на `edulearning@yandex.ru` через FormSubmit.

При первой отправке FormSubmit пришлёт на эту почту письмо активации. Необходимо открыть письмо и подтвердить адрес, после чего следующие заявки будут автоматически приходить на почту.


## Маршруты

| Маршрут | Страница |
| --- | --- |
| `#/` | Главная |
| `#/courses` | Каталог курсов |
| `#/course/:id` | Страница курса |
| `#/directions` | Направления |
| `#/teachers` | Преподаватели |
| `#/reviews` | Отзывы |
| `#/contacts` | Контакты и запись |

## Локальный запуск

Можно открыть `index.html` напрямую в браузере или запустить любой статический сервер.

Пример с Python:

```bash
python -m http.server 5173
```

После запуска сайт будет доступен по адресу:

```text
http://127.0.0.1:5173/
```
